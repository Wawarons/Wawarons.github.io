void com.example.JO2024.JwtAuthenticationFilterTest.test_valid_refresh_token_generates_new_jwt ( ) throws IOException
inline

Test case to verify that a valid refresh token generates a new JWT token.

This method sets up the necessary mocks and data to simulate a valid refresh token scenario. It mocks the necessary dependencies such as Claims, HttpServletResponse, UserDetails, and RefreshToken. The method then verifies that a new JWT token is generated when a valid refresh token is provided. It uses ReflectionTestUtils to inject the necessary mocks into the JwtAuthenticationFilter. Finally, it asserts that the generated cookie contains the expected token name and value.
