void com.example.JO2024.Service.ResetPasswordTokenServiceTest.test_refresh_token_service_get_reresh_token_by_customer_id ( ) throws Exception
inline

Test method to verify the functionality of getting a refresh token by customer id in the ResetPasswordTokenService.

Steps:

  1. Retrieve a customer by email address.
  2. Create a new ResetPasswordToken with the customer, expiration time, and token.
  3. Call the createResetTokenPassword method in the ResetPasswordTokenService to save the token.
  4. Retrieve the saved token using the customer id.
  5. Assert that the IDs of the original and retrieved tokens match.
