Detailed Description

Represents a Data Transfer Object (DTO) for a ticket. This DTO includes information such as the ticket ID, title, description, quantity, price, availability status, start date, and end date. The class provides constructors to create a TicketDTO object with different combinations of parameters. It also includes getter and setter methods for each field, as well as a builder for easy object creation.
