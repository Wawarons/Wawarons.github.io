ResponseEntity<?> com.example.JO2024.controllers.AdminController.createBundle ( @RequestBody BundleDTO bundleDto)
inline

Creates a new bundle based on the provided BundleDTO. Retrieves the ticket using the ticketId from the BundleDTO. Constructs a new Bundle object with the details from the BundleDTO and the retrieved ticket. Saves the created bundle using the bundleService. Returns a ResponseEntity with a success message and HTTP status code.

Parameters
bundleDtoThe BundleDTO containing the details of the new bundle.
Returns
A ResponseEntity with a success message and HTTP status code.
Exceptions
NoSuchElementExceptionif the ticket with the specified ID is not found.
