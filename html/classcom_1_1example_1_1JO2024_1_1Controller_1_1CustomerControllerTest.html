Test case to verify the behavior when attempting to reset a password without a valid token in the database.

Steps:

  1. Create a new PasswordDTO object with a valid password.
  2. Retrieve a customer by email address.
  3. Generate a reset password token using JwtGenerator.
  4. Mock the HttpServletRequest and set the Authorization header with the generated token.
  5. Assert that an InvalidTokenException is thrown when attempting to reset the password.

Test case to verify the successful reset of a customer's password.

Steps:

  1. Create a new PasswordDTO object with a valid password.
  2. Retrieve a customer by email address.
  3. Generate a reset password token using JwtGenerator.
  4. Create a new ResetPasswordToken object and set necessary attributes.
  5. Save the reset password token using ResetPasswordTokenService.
  6. Perform a POST request to reset the password using MockMvc.
  7. Verify that the response contains "Reset password successful".
  8. Retrieve the updated customer and assert that the password has been updated successfully.

Exceptionif an error occurs during the test execution
