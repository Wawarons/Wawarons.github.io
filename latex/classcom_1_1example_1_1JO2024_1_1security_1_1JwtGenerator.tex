\doxysubsection{Detailed Description}
The \doxylink{classcom_1_1example_1_1JO2024_1_1security_1_1JwtGenerator}{Jwt\+Generator} class is responsible for generating and managing JSON Web Tokens (JWTs) for user authentication. It provides methods for generating a JWT, granting token access, creating refresh tokens, and validating JWTs.
