Test case to verify that a valid refresh token generates a new JWT token.

This method sets up the necessary mocks and data to simulate a valid refresh token scenario. It mocks the necessary dependencies such as Claims, Http\+Servlet\+Response, User\+Details, and Refresh\+Token. The method then verifies that a new JWT token is generated when a valid refresh token is provided. It uses Reflection\+Test\+Utils to inject the necessary mocks into the Jwt\+Authentication\+Filter. Finally, it asserts that the generated cookie contains the expected token name and value.
