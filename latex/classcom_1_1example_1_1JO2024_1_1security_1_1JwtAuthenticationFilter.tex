\doxysubsection{Detailed Description}
\doxylink{classcom_1_1example_1_1JO2024_1_1security_1_1JwtAuthenticationFilter}{Jwt\+Authentication\+Filter} is a class that extends Once\+Per\+Request\+Filter and is responsible for filtering and authenticating requests based on JWT tokens. It retrieves the JWT token from a cookie and validates it using the \doxylink{classcom_1_1example_1_1JO2024_1_1security_1_1JwtGenerator}{Jwt\+Generator}. If the token is valid, it retrieves the user details from the \doxylink{classcom_1_1example_1_1JO2024_1_1security_1_1CustomUserDetailsService}{Custom\+User\+Details\+Service} and sets the authentication in the Security\+Context\+Holder. If the token is not valid or not present, it allows the request to proceed without authentication.
